# space-igames-journal

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以静态形式存在，不依赖特定后端服务，适用于个人资料整理或小型项目展示。

## 项目简介

`space-igames-journal` 是一个以 HTML 页面为主的归档仓库。仓库中的页面彼此独立，内容涵盖各类交互式或信息型页面，可作为个人记录、实验或小型工具使用。所有页面均为纯前端实现，可直接在浏览器中打开运行。

## 目录说明

```
.
├── index.html          # 仓库入口页面（可选）
├── pages/              # 存放各类独立 HTML 页面
│   ├── example.html    # 示例页面
│   └── ...
└── assets/             # 静态资源（样式、脚本、图片等）
    ├── css/
    ├── js/
    └── images/
```

- `pages/`：核心目录，每个 HTML 文件为一个独立页面。
- `assets/`：存放页面共用的资源文件，可根据需要组织子目录。

## 页面归档说明

- 每个页面应尽可能自包含，避免跨文件依赖。
- 如需引用外部资源（如字体库、图标库），建议使用 CDN 链接。
- 页面标题、描述等元信息建议在 `<head>` 中明确标注。
- 本仓库不保证所有页面在不同浏览器或设备上的完全兼容性，但会尽量采用通用技术。

## 维护说明

- 欢迎提交新页面或改进现有页面。请确保新增页面不包含恶意代码或侵犯他人权益的内容。
- 提交前请检查页面是否能在本地正常打开，并尽量保持目录结构清晰。
- 如需修改已有页面，请在 PR 中说明改动原因。
- 本仓库由维护者不定期更新，不承诺固定发布周期。
